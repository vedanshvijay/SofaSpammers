# Office IP Mess Installation Guide

## Prerequisites
1. Python 3.8 or higher installed on your computer
2. Internet connection for initial setup

## Installation Steps

1. **Install Python**
   - If you don't have Python installed, download it from https://www.python.org/downloads/
   - During installation, make sure to check "Add Python to PATH"

2. **Install the Application**
   - Double-click the `install_dependencies.py` file
   - This will automatically install all required packages

3. **Run the Application**
   - Double-click `main.py` to start the application
   - The application will start and you can begin using it

## Troubleshooting
- If you encounter any issues during installation, please contact your system administrator
- Make sure your firewall allows the application to communicate on the network
- If you get any Python-related errors, try running the application from Command Prompt:
  1. Open Command Prompt
  2. Navigate to the application folder
  3. Type: `python main.py`

## Support
For any issues or questions, please contact your system administrator.
